## Part 1 God's Vision 要点

Part 1: God’s Vision（第1部分：神的异象）是整本《Jesus Training》手册的理论基础，共两课：

### 1.1 What is God’s Vision?（神的异象是什么？）  
这一课的核心要点如下（提炼自手册整体脉络与目录定位）：

1. 神的终极异象：使“地被神的荣耀充满”（哈巴谷书 2:14），就是“地的每一处都充满对祂的认识，如同水充满洋海”（以赛亚书 11:9）。  
2. 实现这个异象的唯一途径：福音传到万民（万族），马太福音 24:14 被称为“终结经文”——“这天国的福音要传遍天下，作万民的见证，然后末期才来到”。  
3. 当前的拦阻：还有大量“未得之民”（unreached people groups），他们仍活在属灵黑暗中，没有工人、没有教会、没有听过福音。  
4. 神的心意：祂不愿一人沉沦（提后 2:4；彼后 3:9），祂在寻找愿意“站在破口之中”的工人（以西结书 22:30）。  
5. 因此，神的异象不是“教会增长”或“个人得救”那么小，而是“全球覆盖”（global coverage），让所有民族都成为“神的民族”、所有人都成为“神的子民”。  
6. 我们一切事工的唯一动机：唯独神的荣耀（Soli Deo Gloria）。不是建自己的王国，而是让神得着当得的荣耀。

### 1.2 Fulfilling God’s Vision: The Life and Ministry of Jesus（耶稣如何成就神的异象）  
这一课的核心要点：

1. 耶稣一生只有一个目标：完成父交给祂的任务，使父得荣耀（约翰福音 17:4）。  
2. 耶稣自己宣告最大拦阻：“要收的庄稼多，做工的人少”（路加福音 10:2）。  
3. 耶稣的解决方案：  
   - 不是自己一个人跑遍全世界，  
   - 而是把生命投入在“训练门徒”上（训练12个，再训练70个，再训练500个……）。  
4. 耶稣的模式：  
   - 拣选 → 同住 → 教导 → 差遣 → 门徒再去造门徒（马太福音 28:19-20）。  
   - 祂要门徒不只成为“信徒”，而要成为“像祂一样的工人”，能做祂所做的，甚至做比祂更大的事（约翰福音 14:12）。  
5. 耶稣的策略就是“工人生工人、门徒生门徒、教会生教会”，最终实现“全球覆盖”。  
6. 因此，今天我们必须完全效法耶稣的生命与模式，才是真正顺服神的异象。

### Part 1 总括一句话：
神的异象 = 地要被祂的荣耀充满  
实现途径 = 福音必须传到每一个未得之民  
最大拦阻 = 工人太少  
耶稣的解决之道 = 用一生训练门徒，让门徒再去训练门徒，直到全球覆盖、神的荣耀完全彰显。

这就是整个 Jesus Training 手册后面所有策略、方法、步骤的根基——一切都源于 Part 1 对“神的异象”与“耶稣如何成就这异象”的理解。

## 1.2 Fulfilling God’s Vision 完整翻译

### 1.2 成就神的异象：耶稣的生命与事工  
（Fulfilling God’s Vision: The Life and Ministry of Jesus）

#### 耶稣一生只有一个目标  
耶稣在地上的一生只有一个终极目标：**完成父神交给祂的任务，使父得着荣耀**。  
祂在客西马尼园的祷告中清楚地说：  
「父啊！时候到了，愿你荣耀你的儿子，使儿子也荣耀你……我已经荣耀你在地上，把你交给我的事工我已经成全了。」（约 17:1,4）

耶稣把「使父神得荣耀」当作生命的第一优先。祂没有建立自己的王国，也没有追求个人的名声，祂的全部心思就是完成父神交给祂的大使命——让全地都被神的荣耀充满。

#### 耶稣看见的最大拦阻：工人太少  
耶稣清楚地指出，拦阻神国度快速扩展的最大问题不是罪人太多，也不是魔鬼太强，而是：  
「要收的庄稼多，做工的人少。」（路 10:2；约 4:35）

耶稣环顾世界，看见无数灵魂正处于永恒的危险中，却没有人去告诉他们福音。祂心里难过地说：「庄稼已经熟了，可以收割了！」然而工人极其缺乏。

#### 耶稣的解决之道：用一生来训练门徒  
耶稣没有选择自己一个人跑遍全世界（虽然祂有能力这样做），而是把生命中最宝贵的时间（三年半的事奉时间）几乎全部投入在**训练门徒**上。

1. **祂拣选门徒**（可 3:13-14）  
   「耶稣上了山，随自己的意思叫人来……就设立十二个人，要他们常和自己同在，也要差他们去传道。」

2. **祂与门徒同住、同吃、同行、同工**（可 3:14；约 15:27）  
   耶稣让门徒24小时跟在身边，看祂如何祷告、如何传福音、如何医治、如何赶鬼、如何爱人、如何面对逼迫……让门徒「看祂的样式」。

3. **祂教导他们、差遣他们、授权他们**（太 10章；路 9章；路 10章）  
   耶稣先教导他们，然后差遣70个门徒两个两个出去实习，让他们亲手传福音、医病、赶鬼。回来后，耶稣再纠正、教导、再差遣。

4. **祂把使命交给门徒，让门徒继续生养门徒**（约 20:21；太 28:18-20）  
   「父怎样差遣了我，我也怎样差遣你们。」  
   「你们要去，使万民作我的门徒……我就常与你们同在，直到世界的末了。」

耶稣的策略就是：**门徒生门徒、工人生工人、教会生教会**，通过几何级数的倍增，最终让福音传遍天下。

#### 耶稣要门徒成为「像祂一样的人」  
耶稣对门徒的要求从来不是「只要得救就好」，而是：  
「信我的人，要作我所作的事，并且要作比这更大的事。」（约 14:12）

祂要门徒不只是「信徒」（believer），而是「像耶稣一样的工人」——  
- 充满圣灵  
- 遵行神的旨意  
- 为失丧的灵魂摆上生命  
- 能够训练别人也成为工人

#### 耶稣已经为我们立下完美的榜样  
耶稣用自己的生命证明：只要完全效法祂的模式，就一定能快速成就神的异象。  
- 祂从12个门徒开始  
- 到70个  
- 到500多个见证人（林前 15:6）  
- 到五旬节一天3000人  
- 到使徒行传里「耶路撒冷、犹太全地、撒玛利亚，直到地极」都听见福音  
- 最终在使徒时代，保罗可以宣告：「福音传遍了天下」（西 1:6,23）

这一切的起点，就是耶稣把生命投入在「训练门徒」上。

#### 今天我们必须完全回到耶稣的模式  
如果我们想要看见神的异象（全地被祂的荣耀充满）在这一代实现，就只有一个办法：  
**像耶稣一样，把生命投入在训练门徒、差遣工人上，让受训的门徒再去训练更多门徒，直到未得之民全部听见福音，直到神的荣耀覆盖全地。**

这正是《Jesus Training》整本手册的根基：  
我们不只是要自己传福音，我们的使命是——**用耶稣的方式，训练出千千万万像耶稣一样的工人**，让他们去完成耶稣没有完成的那部分任务（即：把福音传到每一个尚未听见的民族）。

正如耶稣所说：  
「你们求庄稼的主多打发工人去收他的庄稼。」（路 10:2）

当我们真正像耶稣一样祷告、像耶稣一样训练、像耶稣一样差遣时，  
神的异象就必在这世代快速成就，  
全地都要被主的荣耀充满！哈利路亚！

## 耶稣如何训练门徒？

根据《Jesus Training Trainer Manual》（以及圣经中耶稣的实际做法），耶稣训练门徒的核心模式可以归纳为以下**8个关键步骤**。这也是整本手册反复强调、要求所有训练者必须完全效法耶稣的模式：

### 耶稣训练门徒的8步模式（耶稣一生都在做的）

1. **祷告寻求神的拣选**  
   耶稣「整夜祷告」之后，才上山呼召祂要的人（路6:12-13）。  
   → 不是随便找人，而是求庄稼的主亲自指明「好土、忠心的人」。

2. **呼召他们常与自己同在**（同住、同行、同工）  
   「就设立十二个人，要他们常和自己同在」（可3:14）。  
   耶稣让门徒24小时跟在身边，看祂怎么祷告、怎么传福音、怎么医治、怎么赶鬼、怎么爱人、怎么面对逼迫……  
   → 训练的第一步不是上课，而是「与耶稣同生活」，让他们先看见活生生的榜样。

3. **用神的话语持续教导他们**  
   耶稣用三年时间不停解释圣经、讲解天国道理、回答问题、纠正错谬（太5-7章山上宝训、无数的比喻、私下讲解……）。  
   → 门徒的生命必须建立在「住在耶稣的话里」（约8:31；15:7）。

4. **示范给他们看**（我做，你们看）  
   耶稣先自己做：传道、医病、赶鬼、怜悯人、饶恕人……门徒在旁边看。  
   → 手册反复强调：训练者必须先「示范」（Demonstration），不能只讲理论。

5. **差遣他们去实践**（你们去做，我看着）  
   先差十二使徒（路9章），再差七十个人两个两个出去（路10章）。  
   给他们权柄，让他们亲手传福音、医病、赶鬼，实际经历神的大能。

6. **让他们回来汇报、纠正、再次教导**  
   七十个人欢欢喜喜地回来：「主啊！因你的名，连鬼也服了我们！」  
   耶稣立刻纠正他们的焦点：「不要因鬼服了你们就欢喜，要因名字记录在天上欢喜」（路10:17-20）。  
   → 这就是手册里「Check（检查作业）→ Teach → Inspect」的循环。

7. **赐下圣灵，使他们有能力**  
   复活后耶稣向他们吹气：「你们受圣灵」（约20:22），五旬节圣灵浇灌（徒2章）。  
   → 没有圣灵，就没有能力生殖繁衍。

8. **交给他们大使命，让他们去生养更多门徒**  
   「天上地下所有的权柄都赐给我了。所以你们要去，使万民作我的门徒……我就常与你们同在，直到世界的末了」（太28:18-20）。  
   「父怎样差遣了我，我也怎样差遣你们」（约20:21）。  
   → 耶稣的终极目标：不是自己跑遍全世界，而是门徒再生门徒、工人生工人、教会生教会，最终实现全球覆盖。

### 一句话总结耶稣的训练策略：
**「我来做，你们看 → 你们去做，我看 → 你们去教别人也这样做」**  
这正是保罗后来写的：「你们该效法我，像我效法基督一样」（林前11:1）。

《Jesus Training》手册把这个模式变成了今天的操作流程：
拣选 → 同住训练 → 教导真理 → 示范福音 → 带他们实践 → 检查纠正 → 差遣出去 → 让他们再去训练别人……直到未得之民全部听见福音，直到全地被神的荣耀充满。

所以，当我们今天问「耶稣如何训练门徒？」答案就是：  
**祂没有开神学院、没有办主日学、没有只讲课，祂把生命完全投入在「与门徒同生活 + 示范 + 差遣 + 再训练别人」上**。  
这也是《Jesus Training》要求我们今天必须100%效法的唯一模式。

   他用20多年时间建立了倍增的教会网络，最终可以宣告：「从耶路撒冷直转到以利哩古，各处的福音都传开了」（罗15:19）。

- 今天我们要快速覆盖全球，唯一的方法就是：**像耶稣训练门徒 → 像保罗效法耶稣一样训练门徒 → 我们今天也必须这样训练**。
- 保罗证明了：耶稣的模式不是只适用于巴勒斯坦，而是放诸四海皆准、可以复制到外邦人的模式。

| 8    | 让他们去生养更多门徒           | 彼得最核心的门徒训练命令在彼得后书： <br>「马可，我的儿子」（彼后5:13）——传统认为马可是彼得的属灵儿子，马可福音就是彼得口述的 <br>彼得训练出马可、西拉（西拉就是后来陪保罗第二次旅行布道的同工，徒15:40） |

### 彼得最强调的门徒繁殖经文（手册也常引用）

彼得后书 1:3-11 是彼得对门徒训练的完整总结：
- 神已赐给我们一切所需（v.3）
- 你们要……更加殷勤，……在信上加上德行……知识……节制……忍耐……敬虔……爱弟兄……爱众人的心（v.5-7）
- 你们若行这些事，永不失脚（v.10）
- 彼得说自己快要去世了，所以要「尽心提醒你们」「叫你们在我去世之后时常记念这些事」（v.12-15）

这正是彼得版的提摩太后书2:2——把从耶稣、从自己领受的真理，不断交托给下一代。

### 彼得训练的果效（圣经 + 历史记载）

- 五旬节一天3000人（徒2:41）
- 后来5000人（徒4:4）
- 撒玛利亚复兴（徒8）
- 外邦人教会开启（徒10）
- 训练出马可（写马可福音）、西拉（陪保罗走第二次宣教旅程）
- 他的讲道和书信至今仍在训练千千万万的门徒

### 《Jesus Training》手册对彼得的定位

虽然手册主要用耶稣和保罗做榜样，但彼得被视为：
- 第一个站出来执行耶稣大使命的人（五旬节）
- 第一个把福音带给外邦人的使徒（哥尼流）
- 耶稣亲自恢复并三次嘱咐「喂养我的羊」的门训对象（约21:15-17）

一句话总结：
**彼得没有发明新方法，他就是耶稣三年半亲手训练出来的“成品门徒”，耶稣走后，彼得原原本本把耶稣的模式继续做下去：公开传道 + 教会中同心教导 + 按手赐圣灵 + 差遣工人 + 写信继续牧养 + 训练属灵儿子（马可），直到为主殉道。**

所以在《Jesus Training》的框架里，彼得不是“另一种方法”，而是耶稣训练模式的第一个成功案例和忠实执行者。今天我们要学的，就是耶稣 → 彼得/保罗 → 我们，一脉相承的同一个门徒训练模式。

## 早期教会门徒倍增策略

根据《Jesus Training》手册和新约圣经（使徒行传 + 书信），早期教会（公元30–100年）实际执行的门徒繁殖策略**完全就是耶稣亲自示范、然后由彼得、保罗、腓力等使徒100%复制的模式**。这也是手册反复强调的“唯一快速覆盖全球的策略”。

### 早期教会门徒繁殖的8大核心策略（圣经实证 + 手册总结）

| 策略 | 圣经实例 | 实际倍增效果 |
|------|----------|--------------|
| 1. 圣灵突然大规模浇灌 + 即时大胆传福音 | 五旬节圣灵降临 → 彼得一次讲道3000人得救（徒2:41）<br>哥尼流家圣灵直接浇灌（徒10:44-46） | 一天3000、一周5000（徒4:4），瞬间从120人暴增到几千人 |
| 2. 所有信徒都是见证人，人人传福音（不是只有传道人） | 「那些分散的人往各处去传道」（徒8:4）<br>「主每日加添人数」（徒2:47） | 逼迫反而造成福音爆炸式扩散：耶路撒冷 → 犹太 → 撒玛利亚 → 安提阿 → 小亚细亚 |
| 3. 立刻建立简单、家型的教会（不是先盖教堂） | 「他们天天在殿里，并在家中擘饼、祷告、教导」（徒2:46；5:42；20:20） | 教会像病毒一样在家庭、社区快速复制 |
| 4. 快速识别并差遣“忠心的人”去开荒 | 腓利去撒玛利亚（徒8）<br>保罗与巴拿巴被圣灵分别出来（徒13:2-3） | 一个腓利就让整个撒玛利亚复兴 |
| 5. 带职训练：边做工边训练（Tentmaker + 同工团队） | 保罗做帐棚养活自己和同工（徒18:3；20:34）<br>带着西拉、提摩太、路加、提多一路训练 | 保罗三次旅行布道中，一边传福音一边训练下一代工人 |
| 6. 四代门徒繁殖链（提摩太后书2:2）<br>这是早期教会最核心的倍增公式 | 耶稣 → 彼得/保罗 → 提摩太/马可/西拉 → 忠心能教导别人的人 → 更多人 | 保罗写信时已经可以宣告「福音传遍天下」（西1:6,23） |
| 7. 写信 + 差派同工继续牧养、纠正、训练 | 保罗写了13封书信<br>差遣提摩太去哥林多、帖撒罗尼迦<br>差遣提多去克里特 | 远距离也能继续门训、纠正假师傅、设立长老（多1:5；提前5:17-22） |
| 8. 目标始终是“到地极”：不满足于已得之地 | 保罗说：「从耶路撒冷直转到以利哩古，到处都传开了……现在我没有余地了，要往西班牙去」（罗15:19-24） | 不到30年，福音就从耶路撒冷传到罗马帝国首都，甚至计划到西班牙 |

### 早期教会惊人的倍增速度（历史事实）

- 公元30年五旬节当天：3000人（徒2:41）

**不到70年，从0个教会 → 覆盖当时已知世界的大部分地区！**
