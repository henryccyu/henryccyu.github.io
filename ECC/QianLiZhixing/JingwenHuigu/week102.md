## 啟示錄 經文回顧 (1): 圓滿的福氣 - 謙理之行

以下是视频的几个核心要点：

### 1. 阅读预言书的挑战与目的

* **预言难以理解**：像《启示录》这样的预言书充满异象和预言，对人来说难以理解，因为我们常常需要经历过后才能明白它的具体所指 [[00:39](http://www.youtube.com/watch?v=fF31kivqg9U&t=39)]。
* **神赐下预言的心意**：神赐给我们的意念是赐平安的意念，而不是降灾祸的意念 [[03:21](http://www.youtube.com/watch?v=fF31kivqg9U&t=201)]。无论异象多么可怕，我们都要相信神对祂儿女有美好的心意 [[03:30](http://www.youtube.com/watch?v=fF31kivqg9U&t=210)]。
* **正确的态度**：我们应当放下担忧和烦恼，去做好现在该做的事情，相信神掌管一切 [[03:48](http://www.youtube.com/watch?v=fF31kivqg9U&t=228)]。

### 2. 阅读《启示录》的三个重点

在阅读《启示录》时，要紧记以下三点：

| 重点 | 核心内容 | 时间戳 |
| :--- | :--- | :--- |
| **神的目的** | **救赎和悔改**：神揭示奥秘的目的是为了让不信的人认识祂，让信祂的人单单依靠祂。在给七个教会的信息中，核心呼召是“悔改” [[04:29](http://www.youtube.com/watch?v=fF31kivqg9U&t=269)]。 | [[05:04](http://www.youtube.com/watch?v=fF31kivqg9U&t=304)] |
| **神的心意** | **坚强战立**：不是为了让人趋吉避凶、逃避灾害，而是要我们在末世世代不惧怕，坚强战立，为主作见证。经文提到要作“得胜的”（conquer/overcome） [[05:37](http://www.youtube.com/watch?v=fF31kivqg9U&t=337)]。 | [[06:03](http://www.youtube.com/watch?v=fF31kivqg9U&t=363)] |
| **我们的心态** | **专注神本身**：预知未来不等于安全感，反而常带来忧虑和惧怕。我们不应纠结于不影响我们或他人认识神公义怜悯的答案，也不应出于逃避或炫耀知识的动机去追求奥秘 [[06:38](http://www.youtube.com/watch?v=fF31kivqg9U&t=398)]。 | [[07:03](http://www.youtube.com/watch?v=fF31kivqg9U&t=423)] |

### 3. 阅读《启示录》的三个重要背景

| 背景 | 核心内容 | 时间戳 |
| :--- | :--- | :--- |
| **数字“七”** | **圆满与甚好**：数字“七”代表着一个完满、成熟，是属神计划的“甚好”的完结。这意味着《启示录》记录的是神计划达到**圆满的福气** [[08:37](http://www.youtube.com/watch?v=fF31kivqg9U&t=517)]。 | [[09:01](http://www.youtube.com/watch?v=fF31kivqg9U&t=541)] |
| **作者约翰的神学思想** | **恩典与慈爱**：作者使徒约翰强调神公义、圣洁与恩典、慈爱的完美结合。即使在最终的大审判中，神的慈爱和恩典依然存留 [[09:50](http://www.youtube.com/watch?v=fF31kivqg9U&t=590)]。 | [[10:30](http://www.youtube.com/watch?v=fF31kivqg9U&t=630)] |
| **约翰与主耶稣的关系** | **敬畏与震撼**：约翰是与主关系极其亲密的人，但当他在异象中看到复活、荣耀、威严的再来之主时，依然仆倒在地“像死了一样”。我们应当带着这种敬畏之心阅读此书 [[17:06](http://www.youtube.com/watch?v=fF31kivqg9U&t=1026)]。 | [[17:49](http://www.youtube.com/watch?v=fF31kivqg9U&t=1069)] |

### 4. 贯穿全书的“七个有福了”

“七个有福了”贯穿《启示录》，代表神对儿女完全圆满的恩典和慈爱 [[10:34](http://www.youtube.com/watch?v=fF31kivqg9U&t=634)]：

1.  **传讲和遵行**：念这书上预言的，和那些听见又遵守其中所记载的，都是有福的 [[01:05:54](http://www.youtube.com/watch?v=fF31kivqg9U&t=3954)]。
2.  **忠心至死**：从今以后，在主里面而死的人有福了（指忠心持守信仰直到生命终点的人） [[01:11:18](http://www.youtube.com/watch?v=fF31kivqg9U&t=4278)]。
3.  **警醒看守**：那警醒看守衣服，免得赤身而行，叫人见到他羞耻的有福了 [[01:15:47](http://www.youtube.com/watch?v=fF31kivqg9U&t=4547)]。
4.  **赴羔羊婚宴**：凡被邀请去赴羔羊的婚宴的有福了 [[01:21:40](http://www.youtube.com/watch?v=fF31kivqg9U&t=4900)]。
5.  **有分第一次复活**：在头一次复活有分的有福了，第二次的死在他们身上没有权柄 [[01:25:22](http://www.youtube.com/watch?v=fF31kivqg9U&t=5122)]。
6.  **遵守预言**：看哪，我必快来！凡遵守这书上预言的有福了 [[01:28:43](http://www.youtube.com/watch?v=fF31kivqg9U&t=5323)]。
7.  **洗净衣服**：那些洗净自己衣服的有福了，他们可得权能到生命树那里，也能从那门进城（指持续悔改，预备圣洁的生命迎接主） [[01:31:37](http://www.youtube.com/watch?v=fF31kivqg9U&t=5497)]。

《启示录》对当时的读者而言，不是可怕的，而是充满**安慰、提醒、力量和盼望**的 [[01:19:01](http://www.youtube.com/watch?v=fF31kivqg9U&t=4741)]。
