## 彼得前、後書 經文回顧：逼迫中的榮耀 - 谦理之行

### 一、 彼得前书（应对外部逼迫）

| 主题 | 圣经书信信息 | 彼得的生命里程碑对照 |
| :--- | :--- | :--- |
| **基督是磐石** | 教会不是建立在彼得这个人身上，而是建立在他所宣告的**信仰**（“你是基督，永生神的儿子”）上。基督是**活石**（房角石），信徒是活石被建造成属灵的居所。 | 主耶稣为他改名“彼得”（石头/磐石），象征他要从摇摆不定的人被塑造成稳固的磐石。 |
| **受苦的荣耀** | 劝勉信徒在没有亲眼见过主的情况下，仍然因信祂而有**“说不出来、满有荣光的大喜乐”** [[08:35](http://www.youtube.com/watch?v=VlgnFNlViI8&t=515)]。 | 彼得曾亲眼见证主耶稣在**变像山上的威荣**，但他学会了谦卑，明白信仰的根基不在眼见，而在信心和爱心的回应。 |
| **效法受苦** | 教导信徒：不要为作恶受苦，而要**为义、为基督**受苦。不要抵挡，而是**顺服**（即便是恶的掌权者）；不要逃避，而是**勇敢面对**；不要向世界**妥协**，而是持守**圣洁**。 | 彼得亲身经历了在逼迫中的失败（三次不认主），他学会了不再靠血气，而是靠圣灵活出圣洁和勇气 [[12:06](http://www.youtube.com/watch?v=VlgnFNlViI8&t=726)]。 |
| **天国子民身份** | 提醒信徒有**四重尊贵身份**：被拣选的族类、君尊的祭司、圣洁的国度、属神的子民。 | 主耶稣恢复彼得使命，要求他以爱心**牧养群羊** [[13:36](http://www.youtube.com/watch?v=VlgnFNlViI8&t=816)]。彼得用“亲爱的”（beloved）称呼信徒，提醒他们是被神所珍贵的。 |
| **宣扬美德** | 基督徒的使命是**“宣扬那召你们出黑暗入奇妙光明者的美德”** [[16:33](http://www.youtube.com/watch?v=VlgnFNlViI8&t=993)]。 | 彼得被主托付牧养群羊的使命。 |

### 二、 彼得后书（应对内部混乱）

《彼得后书》是彼得殉道前的遗言，主要针对教会内部的危机，即**假教师和异端**的威胁。

* **重点呼吁：** 要在认识主耶稣基督上不断**成长**，坚守使徒的**真道**，并以坚定的信心等候天地更新。
* **激励成长：** 激励信徒要分外殷勤，在信心上加上德行、知识、节制、忍耐、虔敬、爱弟兄的心和爱众人的心，活出敬虔的生命 [[17:02](http://www.youtube.com/watch?v=VlgnFNlViI8&t=1022)]。
* **警惕假教师：** 彼得深知内部混乱的破坏力大于外部逼迫，因此他切切叮嘱信徒要警醒，防备那些否认基督、宣扬放纵情欲、讥诮主再来的假师傅 [[18:25](http://www.youtube.com/watch?v=VlgnFNlViI8&t=1105)]。

### 总结：在逼迫中持守基督

彼得前、后书的核心是呼召基督徒：

1.  **在逼迫中效法基督受苦**的榜样。
2.  **持守基督赐予的宝贵身份**（君尊的祭司、圣洁国民）。
3.  **按身份坚守使命**：宣扬神的美德，殷勤坚守真理，并拒绝假教师，按着大牧者耶稣的样式牧养神所托付的群羊 [[19:43](http://www.youtube.com/watch?v=VlgnFNlViI8&t=1183)]。
